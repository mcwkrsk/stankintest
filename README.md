# Telegram Bot for Students

Telegram-бот для автоматизации студенческих задач: управление домашними заданиями, отображение расписания Военного учебного центра (ВУЦ), подписка на информационные рассылки. Бот поддерживает ролевую модель (студент, староста, администратор) и позволяет добавлять, редактировать и просматривать домашние задания по группам и датам.

## Возможности

- **Домашние задания**: добавление, просмотр, редактирование и удаление записей.
- **Расписание ВУЦ**: отображение занятий на текущую неделю для всех групп (данные загружаются из статического модуля).
- **Подписка на рекламные рассылки**: пользователь может подписаться или отписаться от получения периодических информационных сообщений.
- **Ролевая модель**: разграничение прав на основе ролей (`student`, `elder`, `admin`).
- **Автоматическая рассылка**: фоновая задача отправляет рекламные сообщения подписанным пользователям каждые 2 часа.
- **Многошаговые диалоги**: реализованы через состояния пользователя.

## Технологический стек

- Python 3.9+
- `aiosqlite` — асинхронная работа с SQLite
- `pyTelegramBotAPI` (асинхронная версия) — взаимодействие с Telegram API
- `python-dotenv` — загрузка переменных окружения
- `aiohttp` — HTTP-клиент (используется библиотекой бота)
